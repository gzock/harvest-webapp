インポート

  • テンプレートをお使いください。
  • , (カンマ)区切りのCSVファイルのみサポートしています。
  • 文字コードはUTF-8のみサポートしています。
  • "場所"と"撮影対象"をそれぞれ1つ以上指定してください。
  • 同じ"名前"の"場所"を指定しないでください。
  • "場所"は最大で5階層まで作成することができます。
  • 存在しないparentを指定してもエラーになります。
一行目にヘッダ行として、parent,type,nameを指定して下さい。 
        parent: 紐づく"場所"の名前を指定してください。
        type: "場所"あるいは"撮影対象"を指定してください。
        name: 任意の名前を指定してください。
        ※parentに何も指定しない場合は、プロジェクト直下に作成されます。
CSVの例 
        1.  parent,type,name
        2.  ,場所,拠点1
        3.  ,場所,拠点2
        4.  拠点2,場所,部屋1
        5.  拠点1,撮影対象,無線アクセスポイント1
        6.  部屋1,撮影対象,無線アクセスポイント2
このCSVを使用すると、下記のように作成されます。 
        拠点1
          + 無線アクセスポイント1
        拠点2
          + 部屋1
              + 無線アクセスポイント2
同じ"名前"の"場所"を指定しないでください。 
          1.  parent,type,name
          2.  ,場所,拠点1
          3.  ,場所,拠点2
          4.  拠点1,場所,部屋1 ← 同じ名前
          5.  拠点2,場所,部屋1 ← 同じ名前
          6.  部屋1,撮影対象,無線アクセスポイント
 "場所"は最大で5階層まで作成することができます。 
          1.  parent,type,name
          2.  ,場所,拠点1
          3.  拠点1,場所,1号館
          4.  1,場所,1F
          5.  1F,場所,A側
          6.  A側,場所,部屋1
          7.  部屋1,場所,デスクA ← 6階層目
          8.  部屋1,撮影対象,無線アクセスポイント
 存在しないparentを指定してもエラーになります。 
          1.  parent,type,name
          2.  ,場所,拠点1
          3.  ,場所,拠点2
          4.  拠点1,場所,部屋1 
          5.  ビルA,場所,部屋1 ← ビルAは存在しない
          6.  部屋1,撮影対象,無線アクセスポイント
