インポート

  • テンプレートをお使いください。
  • , (カンマ)区切りのCSVファイルのみサポートしています。
  • 文字コードはUTF-8のみサポートしています。
  • "場所"と"撮影対象"をそれぞれ1つ以上指定してください。
CSVの例 
        1.  parent,type,name
        2.  ,場所,拠点1
        3.  ,場所,拠点2
        4.  拠点2,場所,部屋1
        5.  拠点1,撮影対象,無線アクセスポイント1
        6.  部屋1,撮影対象,無線アクセスポイント2
このCSVを使用すると、下記のように作成されます。 
        拠点1
          + 無線アクセスポイント1
        拠点2
          + 部屋1
              + 無線アクセスポイント2

実行中...しばらくお待ち下さい
